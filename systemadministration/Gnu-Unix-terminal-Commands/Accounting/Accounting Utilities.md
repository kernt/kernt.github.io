This is Edition 6.6.2 of the documentation, updated 19 August 2015 for version 6.6.2 of the GNU Accounting Utilities.

- [Preface](https://www.gnu.org/software/acct/manual/accounting.html#Preface): general information about the accounting utilities
- [ac](https://www.gnu.org/software/acct/manual/accounting.html#ac): print statistics about connect time
- [accton](https://www.gnu.org/software/acct/manual/accounting.html#accton): turns accounting on or off
- [lastcomm](https://www.gnu.org/software/acct/manual/accounting.html#lastcomm): list last commands executed
- [sa](https://www.gnu.org/software/acct/manual/accounting.html#sa): print accounting statistics
- [dump-acct](https://www.gnu.org/software/acct/manual/accounting.html#dump_002dacct): print accounting file in human-readable form
