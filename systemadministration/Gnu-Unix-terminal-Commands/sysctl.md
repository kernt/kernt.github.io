**-A** Zeigt alle Werte in Tabellenform

**view all current kernel parameters invoke the `sysctl`**

`sysctl -a`

**check the current sappiness value**

`sysctl vm.swappiness`

**set a kernel parameter at runtime run the `sysctl` command**

`sysctl -w parameter=value`

**load setting**

`sysctl -p`
