# CAT Examples

_Mit meheren Datein_

`cat file1 file2`

_Line Number anzeigen_

`cat -n filename`

_Leere zeilen unterdrücken_

`cat -s geeks.txt`

_Ausgabe von _file1_ an datei _file2_ unten anhängen._

`cat file1 >> file2`

_Ungekerte ausgabe_

`tac filename`
