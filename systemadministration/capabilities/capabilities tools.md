
|Command|Description|
|---|---|
|[capsh](https://linux-audit.com/system-administration/commands/capsh/)|capsh provides a capability shell wrapper to test Linux capabilities|
|[captest](https://linux-audit.com/system-administration/commands/captest/)|captest performs a set of tests related to capabilities and demonstrates them|
|[filecap](https://linux-audit.com/system-administration/commands/filecap/)|filecap shows available capabilities set on binaries in $PATH or specified directory|
|[firejail](https://linux-audit.com/system-administration/commands/firejail/)|firejail sandboxes applications|
|[getcap](https://linux-audit.com/system-administration/commands/getcap/)|getcap queries the available file capabilities|
|[getpcaps](https://linux-audit.com/system-administration/commands/getpcaps/)|getpcaps shows the available process capabilities|
|[netcap](https://linux-audit.com/system-administration/commands/netcap/)|netcap shows an overview of network-related processes and their capabilities|
|[pscap](https://linux-audit.com/system-administration/commands/pscap/)|pscap shows an overview of processes and their assigned capabilities|
|[setcap](https://linux-audit.com/system-administration/commands/setcap/)|setcap adds or removes available file capabilities|
