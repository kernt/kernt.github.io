> _chezmoi helps you manage your personal configuration files (dotfiles, like ~/.gitconfig) across multiple machines._
> 
> _chezmoi provides many features beyond symlinking or using a bare git repo._

```sh
chezmoi init  
chezmoi add ~/.bashrc  
chezmoi edit ~/.bashrc  
chezmoi diff  
chezmoi -v apply
```
